## 👋 What Is This?

This repo is my **public learning journal** for GenAI development.

Each folder is a standalone experiment that tests a real question about how LLMs work — not a copy of a tutorial, not a demo. I build it, run it, measure it, and document what I find.

I'm a **Data Analyst Consultant** with 6+ years of experience across PMI, Stellantis, Bosch, and Daimler Truck AG. I'm now building GenAI skills the only way that works — by shipping real projects.
